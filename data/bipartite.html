Двудольные графы

Двудольный граф — граф, вершины которого можно разделить на два непересекающихся множества так, что каждое ребро графа соединяет вершины из разных множеств.

Свойства двудольных графов

  • Граф является двудольным тогда и только тогда, когда он не содержит циклов нечётной длины
  • Все деревья являются двудольными графами
  • Все графы с циклом чётной длины являются двудольными

Как проверить граф на двудольность?

Для проверки графа на двудольность можно использовать поиск в глубину (DFS) или поиск в ширину (BFS) с раскраской вершин в два цвета:

  1. Выбираем начальную вершину и красим её в цвет 1
  2. Для каждой смежной вершины:
    • Если вершина не окрашена, красим её в противоположный цвет и рекурсивно обрабатываем
    • Если вершина окрашена в тот же цвет, что и текущая, граф не двудольный

  • Сетевые потоки

Проверка графа на двудольность на C++

Ниже представлена реализация проверки графа на двудольность с использованием BFS:

#include <iostream>
#include <vector>
#include <queue>
using namespace std;

bool isBipartite(vector<vector<int>>& graph) {
    int n = graph.size();
    vector<int> color(n, -1); // -1: не окрашена, 0: цвет 1, 1: цвет 2
    
    for (int i = 0; i < n; i++) {
        if (color[i] == -1) {
            queue<int> q;
            q.push(i);
            color[i] = 0;
            
            while (!q.empty()) {
                int u = q.front();
                q.pop();
                
                for (int v : graph[u]) {
                    if (color[v] == -1) {
                        color[v] = 1 - color[u];
                        q.push(v);
                    } else if (color[v] == color[u]) {
                        return false;
                    }
                }
            }
        }
    }
    
    return true;
}

int main() {
    // Пример двудольного графа
    vector<vector<int>> graph1 = {
        {1, 3},
        {0, 2},
        {1, 3},
        {0, 2}
    };
    
    // Пример недвудольного графа (треугольник)
    vector<vector<int>> graph2 = {
        {1, 2},
        {0, 2},
        {0, 1}
    };
    
    cout << "Граф 1 " << (isBipartite(graph1) ? "является" : "не является") << " двудольным" << endl;
    cout << "Граф 2 " << (isBipartite(graph2) ? "является" : "не является") << " двудольным" << endl;
    
    return 0;
}

Пояснение к коду

Данный код реализует проверку графа на двудольность с использованием BFS. Алгоритм пытается раскрасить граф в два цвета так, чтобы смежные вершины имели разные цвета. Если это невозможно, граф не является двудольным.
